# 股票量化交易工具 - 项目规格说明书

## 1. 项目概述

**项目名称:** StockQuant Pro  
**类型:** 桌面端量化交易辅助工具 + VN.py 实盘交易  
**核心功能:** 股票数据分析、策略回测、信号生成，支持与同花顺软件数据互通，集成VN.py实现实盘交易  
**目标用户:** 散户投资者、量化交易爱好者

---

## 2. 功能规格

### 2.1 核心功能

#### 数据模块
- [x] 实时/历史行情获取（东方财富/TuShare数据源）
- [x] 日线/周线/月线/分钟线数据支持
- [x] 数据本地存储与缓存 (SQLite)
- [x] 同花顺数据导入（.h5/.otd 格式兼容）
- [x] 模拟数据后备 (API不可用时)

#### 分析模块
- [x] 常用技术指标：MA, EMA, MACD, RSI, KDJ, BOLL, CCI, ATR, OBV, WR
- [ ] 自定义指标公式
- [ ] 多股票对比分析
- [ ] 板块联动分析

#### 策略模块
- [x] 经典策略模板：双均线、MACD金叉、突破策略、RSI策略
- [x] 策略回测引擎
- [x] 收益统计：收益率、夏普比率、最大回撤
- [ ] 策略参数优化

#### 交易执行 (VN.py)
- [x] VN.py 交易框架集成
- [ ] 实盘交易接口 (CTP/券商API)
- [ ] 订单管理 (下单/撤单/持仓查询)
- [ ] 账户资金查询

#### 信号模块
- [x] 买入/卖出信号生成
- [ ] 信号提醒（桌面通知）
- [x] 信号记录与复盘

### 2.2 界面设计

- **主窗口:** 行情浏览 + 策略监控 (Flask Web UI)
- **分析窗口:** 指标图表 + 信号标注
- **交易窗口:** 实盘交易控制 (待实现)
